问题根因：你的 npm 配置使用了 `https://registry.npmmirror.com/` 国内镜像，而该镜像没有同步 `@anthropic-ai/claude-code-linux-x64` 这个平台特定的 native 二进制包，导致安装后缺少可执行文件。

解决方法：使用 `--registry=https://registry.npmjs.org/` 指定官方源安装即可。

如果以后还需要更新，可以用这个命令：

nvm use 24 && npm install -g @anthropic-ai/claude-code --registry=https://registry.npmjs.org/
