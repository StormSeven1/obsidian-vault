## 后续编辑与 Visio 转换

1. 使用 diagrams.net/draw.io 打开 `source/*.drawio`，文本、节点和连线均可编辑。
2. Word 中优先插入 `svg/*.svg`；如目标环境对 SVG 支持不好，插入 `png/*.png`。
3. 转 Visio 时优先从 draw.io 打开 `.drawio` 后导出 SVG/PDF，再在 Visio 中插入或重构；复杂接口图建议保留 `.drawio` 作为主源文件。
