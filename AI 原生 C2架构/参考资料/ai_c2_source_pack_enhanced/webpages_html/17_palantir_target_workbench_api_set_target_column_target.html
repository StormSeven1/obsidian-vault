## 核心要点

- Endpoint: PUT /api/v2/targetWorkbench/targets/{targetRid}/setTargetColumn。
- Move a Target into a TargetBoardColumn from an old column。
- newColumnId 默认值：DRAFT, PLAN_DEVELOPMENT, PLANNED, EXECUTION, CLOSED。
- OAuth2 scope: api:target-write。

## 本地摘录 / 结构化正文

Endpoint: PUT /api/v2/targetWorkbench/targets/{targetRid}/setTargetColumn

Purpose: Move a Target into a TargetBoardColumn from an old column.

OAuth2 scope: api:target-write.

Request body: SetTargetColumnTargetRequest with boardRid, newColumnId, baseRevisionId, clientId.

newColumnId is equivalent to a collection column ID. Default values are: DRAFT (Identified target), PLAN_DEVELOPMENT (Prioritized target), PLANNED (In coordination), EXECUTION (In execution), CLOSED (Complete).

Response body: EmptySuccessResponse.

Error examples include Target Not On Target Board, Revision Outdated, and permission denied.
